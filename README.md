# PySekai

(WIP) A new Project Sekai inspired engine for [Sonolus](https://sonolus.com).

## Quick Dev Setup
1. Install [uv](https://docs.astral.sh/uv/)
2. Run `uv sync`
3. Add test chart files to `resources/`
4. Run `sonolus-py dev`
